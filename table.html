Langkah-langkah Membuat Tabel

  1. Buka file HTML dan buat struktur dasar seperti <!DOCTYPE html>, <html>, <head>, dan <body>.
  2. Tambahkan elemen <table> untuk membuat tabel dan gunakan <thead> untuk membuat judul kolom.
  3. Di dalam <thead>, tambahkan baris dengan elemen <tr>, dan setiap kolom dengan elemen <th>.
  4. Gunakan elemen <tbody> untuk menambahkan isi tabel, dan tambahkan baris data dengan <tr> serta kolom dengan <td>.
  5. Tambahkan CSS untuk mengatur tampilan tabel seperti warna latar belakang, warna teks, dan efek hover.
  6. Untuk efek interaktif, tambahkan JavaScript jika diperlukan.


Syntax untuk CSS:

              
      :root {
          --body-color: #201e43; /* Ungu pekat untuk latar belakang */
          --color-white: #EEEEEE; /* Putih keabu-abuan yang lembut */
          --text-color-second: #a591ce; /* Abu-abu terang untuk teks sekunder */
          --first-color: #5a4bb7; /* Ungu tua tegas */
          --first-color-hover: #443791; /* Ungu lebih tua untuk efek hover */
          --text-black: #000000; /* Hitam untuk teks */
      }
      
      body {
          font-family: 'Poppins', sans-serif;
          background-color: var(--body-color);
          color: var(--text-color-second);
          display: flex;
          justify-content: center;
          align-items: flex-start;
          flex-direction: column;
          height: auto;
          margin: 0;
          padding: 20px;
      }
      
      /* Navigasi Bar */
      .navbar {
          width: 100%;
          background-color: var(--first-color);
          padding: 10px;
          text-align: center;
          margin-bottom: 20px;
          border-radius: 5px;
      }
      
      .navbar a {
          color: var(--color-white);
          text-decoration: none;
          margin: 0 15px;
          font-weight: 500;
      }
      
      .navbar a:hover {
          text-decoration: underline;
      }
      
      table {
          width: 100%;
          max-width: 1500px; /* Maksimal lebar tabel */
          border-collapse: collapse;
          background-color: var(--color-white);
          border-radius: 10px;
          overflow: hidden;
          box-shadow: 0 0 20px rgba(0, 0, 0, 0.2);
          margin-bottom: 40px; /* Jarak antara tabel */
      }
      
      th, td {
          border: 1px solid var(--first-color);
          padding: 12px;
          text-align: center;
          transition: background-color 0.3s;
      }
      
      th {
          background-color: var(--first-color);
          color: var(--color-white);
      }
      
      tr:nth-child(even) {
          background-color: #f2f2f2;
      }
      
      tr:hover {
          background-color: var(--first-color-hover);
          color: var(--color-white);
      }
      
      /* Mengubah warna teks tabel menjadi hitam */
      td {
          color: var(--text-black);
      }
      tr:hover td {
          color: var(--color-white);
      }
      
      h2 {
          color: var(--text-color-second);
          margin-bottom: 20px;
          text-align: center;
      }

Syntax untuk Membuat Tabel:

               <table>
                   <thead>
                       <tr>
                           <th>NIM</th>
                           <th>Nama</th>
                           <th>IPK Sem 1</th>
                           <th>IPK Sem 2</th>
                           <th>IPK Sem 3</th>
                           <th>IPK Sem 4</th>
                           <th>IPK Sem 5</th>
                           <th>IPK Sem 6</th>
                           <th>IPK Sem 7</th>
                           <th>IPK Sem 8</th>
                           <th>Jurusan</th>
                           <th>Fakultas</th>
                       </tr>
                   </thead>
                   <tbody>
                       <!-- Tambahkan baris data di sini -->
                       <tr>
                           <td colspan="12">Belum ada data yang tersedia</td>
                       </tr>
                   </tbody>
               </table>
