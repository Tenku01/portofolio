Yang Harus Dipersiapkan:

1. Install Git terlebih dahulu
2. Login akun Github/Membuat Akun github pada website
3. Membuat Repository pada github dan jangan lupa di Public agar dapat dilihat siapa saja            
4. Masuk ke terminal dan arahkan ke path Project anda

Contoh Kode untuk Deploy Project ke GitHub

          
1. Inisialisasi repository Git:
 $ git init

2. Tambahkan file ke staging area:
$ git add .

3. Commit perubahan:
$ git commit -m "Initial commit"

4. Hubungkan repository lokal dengan repository remote:
$ git remote add origin https://github.com/username/repository.git

5. Push ke GitHub:
$ git branch -M main
$ git push -u origin main

Cara Online Web/Project kalian Melalui Github:

1. Buka file project repository yang akan dijadikan Online
2. Lalu Pergi ke Setting
3. Lalu ke menu Pages di samping kiri          
4. Pada bagian "Branch" Ubah dari "None" menjadi "main" lalu klik save.
5. Tunggu beberapa saat dan lalu refresh dan akan muncul link untuk project anda
